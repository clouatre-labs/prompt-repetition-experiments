# Changelog

All notable changes to this dataset repository are listed here.

## [1.0.5] -- 2026-04-27
### Added
- Exp3 (Kotlin grammar synthesis): 10 session logs, scores, analysis, efficiency data
- Schema normalisation: analysis.json backfilled to full canonical schema for all three experiments
- Wilson 95% CIs for all criterion pass rates

### Changed
- Effect size metric: Cohen's d replaced with rank-biserial r (Kerby 2014) across all experiments
- Statistical method: U-test denominator corrected
